Symfony 4 starter kit with this set up:

* FOSUser Bundle
* guard authenticator
* registartion and login form templates
* very basic [Gentelella](https://github.com/puikinsh/gentelella) admin template set up

1. Clone this repo
2. Set up DB
3. Run migration
